Pengertian Link

HTML Link atau Hyperlink adalah Sekumpulan kata,ataupun gambar yg apabila di klik akan menuju documen/halaman lain atai nagiam lain dari document/halaman yg sama

Macam2 link di HTML :
  • Link Relative
  • Link absolute
  • Link Kebagian lain di dalam documen yg sama
  • Link ke email

Link relative

Dibuat apabila kita mebuat suatu link pada page ke page yg lain pada komputer yg sama,tidak memerlukan internet yg lengkap,contoh kita akan membuka file asli,disini Cuy

Link Absoulute

Dibuat apabila kita kita aka membuat link ke page web lain, yg berada diluar website kita dgn cara, sehingga alamat yg kita tulis secara lengkap sesuai dgn alamat web yg kita tuju, Contoh die kuy

Link E-mail

link email digunakan untuk membuat link saat kita akan mengirim email dgn catatan tdk akan bisa menggunakan email client terinstal, contoh kita akan mengirim email ke alamat chandrayeager@gmail.com

Link Kebagian lain di dalam documen yg sama

Link ini digunakan apabila page yg terlalu panjang sering kali kita harus srool untuk menuju bagian yg diinginkan,untuk mempermudah kita gunakan link document ke bagian lain pada halam yg sama:

  1. Membuat id (name) pada bagian yg di tuju
  2. Membuat script link pada tulisan yg kita inginkan

    Kita akan membuat link pada bagian document latihan in pada bagian atas jenis2 link silakan di pelajari scriptnya
